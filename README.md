This is a Windows batch script that automates MP3 processing:

Trims silence at the beginning and end of each track using ffmpeg.

Counts and displays the total number of tracks and shows a live progress counter.

Optional renaming: renames files in the format Song - Artist.

Custom output folder support: save processed tracks to a chosen folder or automatically create a _crop folder.

Perfect for quickly cleaning up a music collection before sharing or listening.
